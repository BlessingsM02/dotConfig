
A clean, modern Wayland-based desktop environment configuration featuring **Noctalia Shell** and **Hyprland**.

---

## Requirements

Below are the primary applications and tools configured and used in this setup:

| Application | Description | Official Link |
| :--- | :--- | :--- |
| **Noctalia Shell** | A sleek, minimal desktop shell designed for Wayland (bars, docks, launchers) | [noctalia.dev](https://noctalia.dev/) |
| **Kitty** | A fast, feature-rich, GPU-accelerated terminal emulator | [sw.kovidgoyal.net/kitty](https://sw.kovidgoyal.net/kitty/) |
| **Zsh** | Powerhouse interactive shell and command language | [zsh.org](https://www.zsh.org/) |
| **Fastfetch** | Light-weight, ultra-fast system information fetch tool | [github.com/fastfetch-cli/fastfetch](https://github.com/fastfetch-cli/fastfetch) |
| **Neovim** | Hyperextensible, Vim-based text editor | [neovim.io](https://neovim.io/) |
